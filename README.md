# 跨境卖家利润计算器

一个简单清晰的开源跨境卖家利润计算器，适合跨境卖家快速估算产品成本、头程、尾程、平台佣金、广告费、支付手续费和建议售价。

## 功能

- 产品成本、头程费用、尾程费用计算
- 平台佣金、支付手续费、广告费率计算
- 目标利润率反推建议售价
- 场景模板快速切换
- 买家价格说明卡片
- 一键截图下载
- 复制图片到剪贴板
- 本地保存输入数据
- Docker 和 1Panel 部署

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## Docker 部署

```bash
docker compose up -d --build
```

访问：

```txt
http://127.0.0.1:8088
```

## 1Panel 反向代理

推荐把容器端口绑定到本机：

```yaml
ports:
  - "127.0.0.1:8088:80"
```

然后在 1Panel 中创建反向代理网站：

```txt
temu.nowdn.com -> http://127.0.0.1:8088
```

## 计算公式

固定成本：

```txt
产品成本 + 头程费用 + 尾程费用 + 仓储费 + 税费 + 退货损耗 + 其他成本
```

变动费率：

```txt
平台佣金 + 支付手续费 + 广告费率
```

建议售价：

```txt
固定成本 / (1 - 变动费率 - 目标利润率)
```
