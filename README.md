# 跨境卖家计算器
<img width="1024" height="1024" alt="image" src="https://github.com/user-attachments/assets/76a2bf09-b12f-41e6-a11e-d5de05124d30" />

一个免费的开源跨境卖家利润计算器，用来快速核算成本、费率、利润和建议售价。

在线使用：<https://temu.nowdn.com/>

## 它主要回答两个问题

1. 我卖这个价，单件能赚多少？
2. 我要达到目标利润率，建议售价应该是多少？

## 适合谁用

- 跨境电商卖家
- 独立站卖家
- Amazon、Temu、TikTok Shop、Shopee 等平台卖家
- 海外仓、代发、铺货和精品选品团队
- 需要给客户或团队快速核对报价的人

## 功能

- 当前售价利润测算
- 目标利润率反推建议售价
- 产品成本、头程费用、尾程费用、仓储费、税费、退货损耗计算
- 平台佣金、支付手续费、广告费率计算
- 活动扣点、达人佣金、其他比例扣款计算
- 其他固定扣款计算
- 保本售价、变动费率、目标售价利润展示
- 成本占比条
- 报价核对工具
- 一键截图下载
- 复制图片到剪贴板
- 汇率参考与自定义汇率
- 场景模板快速切换
- 本地保存输入数据
- Docker 和 1Panel 部署

## 隐私

这个工具不需要登录。计算在浏览器本地完成，不上传你的成本、售价、利润或业务数据。

## 计算逻辑

固定成本：

```txt
产品成本 + 头程费用 + 尾程费用 + 仓储费 + 税费 + 退货损耗 + 其他固定扣款
```

变动费率：

```txt
平台佣金 + 支付手续费 + 广告费率 + 活动扣点 + 达人佣金 + 其他比例扣款
```

当前售价利润：

```txt
当前售价 - 固定成本 - 当前售价 × 变动费率
```

保本售价：

```txt
固定成本 / (1 - 变动费率)
```

建议售价：

```txt
固定成本 / (1 - 变动费率 - 目标利润率)
```

如果变动费率和目标利润率合计达到或超过 100%，工具会提示无法反推合理建议售价。

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## Docker 部署

```bash
docker compose up -d --build
```

访问：

```txt
http://127.0.0.1:8088
```

## 1Panel 反向代理

推荐把容器端口绑定到本机：

```yaml
ports:
  - "127.0.0.1:8088:80"
```

然后在 1Panel 中创建反向代理网站：

```txt
temu.nowdn.com -> http://127.0.0.1:8088
```
